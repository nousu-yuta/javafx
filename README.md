・背景、きっかけ　→　GUIアプリ開発に興味を持ち、ユーザーが実際に使用できるアプリを作成したいと考えた。
授業内での開発（App1〜App5）を通して基礎的な実装力を身につける中で、より実用性の高いアプリを自ら設計・開発したいと考え、ストップウォッチ（App6）の制作に取り組んだ。

・取り組み内容　→　App1〜App5では、仕様に沿ってUI・機能を実装しつつ、自分なりに機能の追加や改善を行った。
App6では、自らテーマを決めて設計から実装まで行い、日常的に使えるシンプルなツールとしてストップウォッチを作成した。
スタートボタン押下で計測を開始し、表示をストップに切り替えることで直感的な操作を実現した。
停止後に再度スタートを押すことで途中から計測を再開できる仕様とし、リセットボタンによりワンクリックで初期状態に戻せるようにした。
JavaFXのTimelineを用いて時間計測を実装し、スタート・ストップの状態を管理することで正確な動作を実現した。

・工夫した点　→　UIの使いやすさを意識し、直感的に操作できる配置を心がけた。
また、誤操作時にも問題が発生しないようエラー処理を実装し、安定した動作を目指した。
また、タイマーの状態（開始・停止）を管理することで、ユーザーの操作に応じた正確な挙動を実現した。

・今後について　→　今回の開発を通して、ユーザー目線での設計の重要性を実感した。
今後は、実際の利用シーンや課題をより具体的に想定し、それに基づいた機能改善やUI/UXの最適化に取り組みたい。
